Групповые чаты

Показывает создание группы и рассылку одного сообщения всем участникам.

Главная Пользователи Личные сообщения Группы Файлы Ключи и аудит

Методы страницы

  • POST /api/messages/group - создание группы с участниками.
  • POST /api/messages/group/{groupId}/send - отправка сообщения в группу.

Укажи owner и участников (по логинам), затем отправь сообщение в группу.

Действия по сценариям

Сценарий A: Создание группы

Создает чат с владельцем и списком участников (по логинам через запятую).

1. Группа создана

Сценарий B: Отправка в группу

Отправляет сообщение в созданную группу и показывает fan-out на участников.

2. Рассылка участникам

Подсказка: сначала выполни сценарий A, затем сценарий B с полученным `groupId`.
