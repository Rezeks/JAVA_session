Крипто-методы и аудит

Визуально показывает операции с ключами и журнал событий безопасности.

Главная Пользователи Личные сообщения Группы Файлы Ключи и аудит

Методы страницы

  • GET /api/keys/{ownerId}/public - экспорт публичного ключа.
  • GET /api/keys/{ownerId}/private - экспорт приватного ключа (демо-режим).
  • POST /api/keys/{ownerId}/import - импорт пары ключей.
  • GET /api/audit - полный список аудита.

Выбери пользователя и выполни операции с ключами, затем обнови аудит.

Действия по сценариям

Сценарий A: Экспорт ключей

Выбери пользователя и получи public/private ключ для демонстрации key-management.

Сценарий B: Импорт ключевой пары

Можно вставить экспортированные ключи обратно, чтобы показать обратную операцию.
