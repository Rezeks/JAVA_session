# ⚡ QUICK START COMMANDS

## Windows (PowerShell)

```powershell
# Перейти в проект
cd "C:\Users\Rezeks\IdeaProjects\JAVA_session"

# Компилировать
mvn -q -DskipTests clean compile

# Запустить
mvn spring-boot:run

# Или просто дважды щёлкнуть run.bat
```

## Linux / Mac

```bash
cd ~/IdeaProjects/JAVA_session

mvn -q -DskipTests clean compile

mvn spring-boot:run

# Или
bash run.sh
```
